Auth Service API

Production-ready authentication microservice built with:

• Node.js
• Express
• TypeScript
• Prisma ORM
• PostgreSQL (NeonDB)
• JWT Authentication
• Refresh Tokens
• Email Verification
• Password Reset
• Rate Limiting
• Swagger API Docs
• Docker Support

Features

- Register
- Email Verification
- Login
- Refresh Token
- Logout
- Get Profile
- Forgot Password
- Reset Password

Run Locally
docker compose up --build

API Docs
http://localhost:5000/api-docs
